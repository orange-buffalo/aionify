# Documentation Images

This directory contains screenshots and images for the documentation.

## Required Screenshots

The following screenshots should be added to this directory:

### api-token-generation.png
Screenshot showing the API Access Token section in Settings page with the "Generate Token" button visible.

Steps to capture:
1. Log in to Aionify
2. Navigate to Settings (gear icon)
3. Scroll to "API Access Token" section
4. Take screenshot showing the full section with "Generate Token" button

### api-token-display.png
Screenshot showing the API Access Token section with a generated token, eye icon, and copy icon.

Steps to capture:
1. Log in to Aionify
2. Navigate to Settings (gear icon)
3. Generate or reveal an API token
4. Click the eye icon to show the token
5. Take screenshot showing the token input field, eye icon, copy icon, and action buttons

## Image Guidelines

- Use PNG format for screenshots
- Crop to show only relevant UI sections
- Include enough context to orient the user
- Ensure text is readable (minimum 12pt equivalent)
- Avoid showing sensitive data (use test accounts)
- Update images when UI changes significantly
